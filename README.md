# Fullstack project

## Install

### Backend

- Install Node.js
  - Recomended version - 14.8 LTS
  - From [website](https://nodejs.org/en/)
  - From [nvm](https://github.com/nvm-sh/nvm) (recomended)
- Install dependencies
  ```bash
  cd backend
  npm install
  ```
- Run server
  ```bash
  npm run start
  ```

### Frontend

- Install Node.js
  - Recomended version - 14.8 LTS
  - From [website](https://nodejs.org/en/)
  - From [nvm](https://github.com/nvm-sh/nvm) (recomended)
- Install dependencies
  ```bash
  cd frontend
  npm install
  ```
- Run server
  ```bash
  npm run dev
  ```
